# MVP Scope

This repository implements the local experiment described in `/Users/maxon/Downloads/SEMANTIC_CORE_CODEX_EXPERIMENT.md`.

The current implementation is deliberately small:

- synthetic RU-EN phrase domain;
- explicit semantic atom vectors;
- compact 128-dimensional semantic bottleneck;
- direct baseline translator;
- semantic translator with nearest prototype decoding;
- no external APIs, no scraping, no chatbot, no UI.

This is a feasibility probe, not proof of a universal semantic system.
